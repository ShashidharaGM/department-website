## 🚀 Project Overview

This is a **Department Website** built using **React (Vite)** as part of a web development project.
It showcases academic information, events, gallery, and user interaction features with a modern UI.

---

## ⚙️ Tech Stack

* ⚛️ React (Vite)
* 🎨 CSS (Custom Styling)
* 🔄 React Router
* 🔔 React Toastify
* 🌐 GitHub Pages (Deployment)

---

## ▶️ How to Run the Project Locally

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/department-website.git
cd department-website
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start development server

```bash
npm run dev
```

👉 Open in browser:

```
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

---

## 🚀 Deploy to GitHub Pages

```bash
npm run deploy
```

## 🌟 Features Available

### 🏠 Home Page

* Welcome section
* Auto-sliding **event carousel (2 events at a time)**
* Achievements section

---

### 📖 About Page

* Department overview
* Vision & Mission
* Key highlights

---

### 🎉 Events Page

* Event filtering (Tech / Cultural)
* Flip card UI
* Event details (date + description)
* Register button → navigates to contact

---

### 🖼️ Gallery Page

* Grid-based image gallery
* Hover overlay with titles
* Lightbox preview on click

---

### 📬 Contact Page

* Form with validation
* Error handling
* Toast notifications

---

### 🏆 Achievements Section

* Highlights of department success
* Clean card-based UI

---

### 🌙 Dark / Light Theme

* Toggle between themes
* Persistent using localStorage
